git init - Iniciando repositorio no git.
git status - Verificando se é preciso subir atualização
git add . - enviar todo conteúdo do diretorio
git commit - m "projeto-dbb" - comitando a atualização

git log - Vendo o log da alteração
git remote add origin https://github.com/marcossilva8428/projeto-dbb.git - conectando o diretorio local com o meu repositorio no github
git push -u origin master - empurrando os arquivos para meu repositorio no git hub
